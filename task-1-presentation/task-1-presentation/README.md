# Task 1 – Digital Literacy Infographic

This task focuses on creating an infographic to spread awareness about digital literacy.

## 📌 Description

An infographic was created using Canva to explain the concept of digital literacy. It includes key topics such as:

- Meaning of digital literacy  
- Useful digital tools for students  
- Safe internet practices  
- Email etiquette  

The infographic was designed using simple icons, headings, and bullet points to make it visually appealing and easy to understand.

## 🎯 Learning Outcome

Through this task, I learned how to present information in a clear and visually effective manner. It also helped me understand the importance of digital literacy and responsible online behavior.
